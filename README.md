# Calendar

App to manage tasks.

## Stack

[Docker](https://www.docker.com/)

[Node.js](https://nodejs.org/en/)

## How to

### Production

```sh
docker run -d -p 80:80 tmvdl/projects:calendar
```

### Development

```sh
bash env/pull.sh 

bash env/install.sh 

bash env/up.sh 
```
