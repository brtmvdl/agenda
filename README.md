# Calendario

Aplicativo para gestão de tarefas e compromissos.

## Stack

[Docker](https://www.docker.com/)

[Node.js](https://nodejs.org/en/)

## Como usar

### Em produção

```sh
docker run -d -p 80:80 tmvdl/projects:calendar
```

### Em desenvolvimento

```sh
bash env/pull.sh 

bash env/install.sh 

bash env/up.sh 
```
